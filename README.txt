Mazzuma Mobile Money Payment Gateway for WooCommerce

== Description ==
Mazzuma Mobile Money Payment Gateway for accepting mobile money payments in your WordPress WooCommerce online shop. Works for mobile money in Ghana only. 

== Installation ==
Go to the WooCommerce plugin settings, and the Payments tab.

If the plugin correctly installed, there should be a Mazzuma payment option among the payment gateways listed. Ensure it is enabled.

Click on the \'Manage\'  for Mazzuma, and you would be taken to the plugin configuration page.

Add the API key generated from the dashboard, and any other appropriate details. Please note payments would not work without this API key. You can get an API key by visiting https://dashboard.mazzuma.com

Once done, hit save.
